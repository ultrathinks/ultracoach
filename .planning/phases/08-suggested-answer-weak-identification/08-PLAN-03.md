<action>
Create a new file `src/entities/feedback/lib.ts` with the following content:

```typescript
import type { QuestionAnalysis } from "./types";

const DEFAULT_WEAK_THRESHOLD = 70;

export function identifyWeakAnswers(
  analyses: QuestionAnalysis[],
  threshold = DEFAULT_WEAK_THRESHOLD,
): QuestionAnalysis[] {
  return analyses
    .filter((qa) => qa.contentScore < threshold)
    .sort((a, b) => a.contentScore - b.contentScore);
}
```

Key details:
- Import `QuestionAnalysis` from `./types` (not from schema, to avoid pulling zod into client bundles)
- Default threshold is 70 (matching CONTEXT.md decision: contentScore < 70 is weak)
- Returns empty array when all scores >= threshold ("No weak answers")
- Sorted ascending by contentScore (worst first)
- No `as` casts, no side effects
</action>
