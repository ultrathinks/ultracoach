# Plan 01: Schema extension — add suggestedAnswer to questionAnalysisSchema

<objective>
Add `suggestedAnswer: z.string().optional()` to the existing `questionAnalysisSchema` in `src/entities/feedback/schema.ts`. This is the foundational change that enables Plan 02 (batch generation) and Plan 03 (weak identification) to operate on typed data. Because `QuestionAnalysis` is defined as `z.infer<typeof questionAnalysisSchema>` in `types.ts`, the type updates automatically with no additional changes. Because the field is `optional()`, all existing v1.1 session feedback data continues to parse without error.
</objective>
