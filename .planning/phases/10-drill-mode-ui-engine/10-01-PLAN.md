# Plan 10-01: Drill engine hook

## Objective

Create `src/features/drill/use-drill-engine.ts` — the core React hook that drives the drill practice loop. It re-uses `createVad()` from `vad.ts` and the MediaRecorder + Whisper transcription pattern from `use-interview-engine.ts`. The hook manages a state machine (`prep | listening | processing | feedback | done`), enforces the 15-word gate (skip LLM if too short), calls the drill API, tracks attempts (max 5), and detects goal achievement (80+ contentScore). Exposes `{ drillPhase, transcript, result, audioLevel, attemptCount, bestScore, goalAchieved, startDrill, stopDrill, cleanup }`.

## must_haves

- [ ] `src/features/drill/use-drill-engine.ts` exists with `export function useDrillEngine(`
- [ ] State machine type `DrillPhase = "prep" | "listening" | "processing" | "feedback" | "done"` exported
- [ ] `DrillResult` type exported with `{ contentScore: number; feedback: string; starFulfillment: { situation: boolean; task: boolean; action: boolean; result: boolean } }`
- [ ] `createVad` imported from `@/features/interview-engine/vad`
- [ ] MediaRecorder + audioChunksRef pattern for recording audio
- [ ] Whisper transcription via `POST /api/whisper` with FormData
- [ ] 15-word gate: transcript word count < 15 skips LLM call, sets validation error
- [ ] Drill API call: `POST /api/sessions/${sessionId}/drill` with `{ questionId, transcript }`
- [ ] Max 5 attempts — after 5th attempt `drillPhase` becomes `"done"`
- [ ] 80+ contentScore sets `goalAchieved = true` and `drillPhase = "done"`
- [ ] `bestScore` tracks highest contentScore across attempts
- [ ] `loopAbortRef` pattern for cleanup on unmount/stop
- [ ] `src/features/drill/index.ts` re-exports `useDrillEngine`, `DrillPhase`, `DrillResult`

**transcribeAudio (useCallback):**
Same pattern as use-interview-engine.ts lines 80-92:
- Create FormData, append audioBlob as "answer.webm"
- POST to "/api/whisper"
- Parse with transcribeResponseSchema
- Return text string

**startDrill (useCallback, depends on [sessionId, questionId]):**
1. `loopAbortRef.current = false;`
2. `setValidationError(null);`
3. `setTranscript("");`
4. `setResult(null);`
5. `setDrillPhase("listening");`
6. `setAudioLevel(0);`
7. Get stream from `streamRef.current` — if null, return early
8. `audioChunksRef.current = [];`
9. Create `audioStream = new MediaStream(stream.getAudioTracks())`
10. Detect mimeType: `MediaRecorder.isTypeSupported("audio/webm;codecs=opus") ? "audio/webm;codecs=opus" : "audio/webm"`
11. Create `mediaRecorder = new MediaRecorder(audioStream, { mimeType })`
12. Set `mediaRecorderRef.current = mediaRecorder`
13. `mediaRecorder.ondataavailable = (e) => { if (e.data.size > 0) audioChunksRef.current.push(e.data); }`
14. `mediaRecorder.start(1000);`
15. Create VAD:
```typescript
vadRef.current = createVad({
  threshold: 0.05,
  silenceDelay: 2500,
  minSpeechDuration: 2000,
  onLevel: (rms) => setAudioLevel(rms),
  onSpeechStart: () => {},
  onSpeechEnd: async () => {
    mediaRecorder.stop();
    vadRef.current?.stop();
    setAudioLevel(0);

    if (loopAbortRef.current) return;

    setDrillPhase("processing");

    const audioBlob = new Blob(audioChunksRef.current, { type: "audio/webm" });

    if (audioBlob.size < 1000) {
      setTranscript("");
      setValidationError("음성이 감지되지 않았습니다");
      setDrillPhase("feedback");
      return;
    }

    try {
      const text = await transcribeAudio(audioBlob);
      setTranscript(text);

      // 15-word gate
      const wordCount = text.trim().split(/\s+/).length;
      if (wordCount < MIN_WORD_COUNT) {
        setValidationError("답변이 너무 짧습니다. 더 구체적으로 답변해 주세요");
        setDrillPhase("feedback");
        return;
      }

      setValidationError(null);

      // Call drill API
      const res = await fetch(`/api/sessions/${sessionId}/drill`, {
        method: "POST",
        headers: { "Content-Type": "application/json" },
        body: JSON.stringify({ questionId, transcript: text }),
      });

      if (!res.ok) throw new Error("drill feedback failed");

      const data = drillResponseSchema.parse(await res.json());

      const newAttempt = attemptCount + 1; // read from ref-like closure
      setAttemptCount(newAttempt);
      setResult(data);

      if (data.contentScore > bestScore) { // compare via closure
        setBestScore(data.contentScore);
      }

      if (data.contentScore >= GOAL_SCORE) {
        setGoalAchieved(true);
        setDrillPhase("done");
      } else if (newAttempt >= MAX_ATTEMPTS) {
        setDrillPhase("done");
      } else {
        setDrillPhase("feedback");
      }
    } catch (err) {
      console.error("drill processing failed:", err);
      setValidationError("분석 중 오류가 발생했습니다. 다시 시도해 주세요");
      setDrillPhase("feedback");
    }
  },
});
vadRef.current.start(stream);
```

**IMPORTANT closure fix:** Because `attemptCount` and `bestScore` can be stale in the `onSpeechEnd` closure, use refs alongside the state:
- Add `const attemptCountRef = useRef(0);` and `const bestScoreRef = useRef(0);`
- In onSpeechEnd, read from `attemptCountRef.current` and `bestScoreRef.current`
- When updating: `attemptCountRef.current = newAttempt; setAttemptCount(newAttempt);` and same for bestScore

**stopDrill (useCallback):**
1. `loopAbortRef.current = true;`
2. `vadRef.current?.stop(); vadRef.current = null;`
3. `mediaRecorderRef.current?.stop(); mediaRecorderRef.current = null;`
4. `setAudioLevel(0);`

**cleanup (useCallback):**
1. Call `stopDrill()`
2. `streamRef.current?.getTracks().forEach((t) => { t.stop(); });`
3. `streamRef.current = null;`

**Return object:**
```typescript
return {
  drillPhase,
  transcript,
  result,
  audioLevel,
  attemptCount,
  bestScore,
  goalAchieved,
  validationError,
  startDrill,
  stopDrill,
  cleanup,
  streamRef,
};
```

Note: `streamRef` is exposed so the drill-screen widget can assign `getUserMedia` stream to it.
