# Plan 10-02: Drill preparation screen

## Objective

Create `src/features/drill/drill-prep-screen.tsx` — the preparation screen shown before a drill attempt starts. It displays a camera preview (getUserMedia), the full question text, a collapsible suggested-answer panel (chevron pattern from `ai-recommendation-card.tsx`), handles permission errors inline, and provides a "연습 시작" button. This component is independent of the drill engine hook — it receives callbacks via props.

## must_haves

- [ ] `src/features/drill/drill-prep-screen.tsx` exists with named export `DrillPrepScreen`
- [ ] Camera preview: `getUserMedia({ video: true, audio: true })` → `<video>` srcObject
- [ ] Full question text displayed
- [ ] Collapsible suggestedAnswer panel (closed by default, chevron rotates 180deg on open)
- [ ] Permission error: inline error message + "다시 시도" button
- [ ] "연습 시작" button calls `onStart(stream)` with the MediaStream
- [ ] Props type: `{ question: string; suggestedAnswer: string | null; onStart: (stream: MediaStream) => void }`

```
"use client";

import { useCallback, useEffect, useRef, useState } from "react";
import { cn } from "@/shared/lib/cn";
```

**Props interface:**

```typescript
interface DrillPrepScreenProps {
  question: string;
  suggestedAnswer: string | null;
  onStart: (stream: MediaStream) => void;
}
```

**Component:**

```typescript
export function DrillPrepScreen({ question, suggestedAnswer, onStart }: DrillPrepScreenProps) {
```

**State & refs:**
- `const webcamRef = useRef<HTMLVideoElement>(null);`
- `const streamRef = useRef<MediaStream | null>(null);`
- `const [permissionError, setPermissionError] = useState<string | null>(null);`
- `const [expanded, setExpanded] = useState(false);`
- `const [mediaReady, setMediaReady] = useState(false);`

**initMedia (useCallback):**
1. `setPermissionError(null);`
2. Try `getUserMedia({ video: true, audio: { echoCancellation: true, noiseSuppression: true, autoGainControl: true } })`
3. On success:
   - `streamRef.current = stream;`
   - `if (webcamRef.current) webcamRef.current.srcObject = stream;`
   - `setMediaReady(true);`
4. On error (catch DOMException):
   - If `name === "NotAllowedError"`: set error "카메라와 마이크 접근이 거부되었습니다. 브라우저 설정에서 권한을 허용해 주세요"
   - If `name === "NotFoundError"`: set error "카메라 또는 마이크를 찾을 수 없습니다"
   - Else: set error "카메라 연결에 실패했습니다"

**useEffect on mount:**
- Call `initMedia()`
- Cleanup: stop all tracks on streamRef.current, set webcamRef.current.srcObject = null

**handleStart callback:**
1. If `streamRef.current` is null, call `initMedia()` and return
2. Call `onStart(streamRef.current)`

**Render structure (dark mode, glassmorphism):**

```tsx
<div className="min-h-[calc(100dvh-4rem)] flex flex-col items-center justify-center px-6 py-10">
  <div className="w-full max-w-2xl space-y-6">
    {/* Question text */}
    <div className="text-center">
      <h1 className="text-xl font-bold leading-relaxed">{question}</h1>
    </div>

    {/* Camera preview */}
    <div className="relative aspect-video w-full rounded-xl overflow-hidden bg-card border border-white/[0.06]">
      <video
        ref={webcamRef}
        autoPlay
        muted
        playsInline
        className="w-full h-full object-cover"
      />
      {!mediaReady && !permissionError && (
        <div className="absolute inset-0 flex items-center justify-center">
          <div className="w-6 h-6 rounded-full border-2 border-foreground/30 border-t-foreground animate-spin" />
        </div>
      )}
      {permissionError && (
        <div className="absolute inset-0 flex flex-col items-center justify-center gap-4 bg-card p-6 text-center">
          <p className="text-sm text-red">{permissionError}</p>
          <button
            type="button"
            onClick={initMedia}
            className="px-4 py-2 rounded-lg text-sm font-medium border border-white/[0.06] hover:bg-card-hover transition-colors"
          >
            다시 시도
          </button>
        </div>
      )}
    </div>

    {/* Suggested answer collapsible panel */}
    {suggestedAnswer && (
      <div
        className="rounded-xl bg-card border border-white/[0.06] p-5 cursor-pointer select-none"
        onClick={() => setExpanded((prev) => !prev)}
        onKeyDown={(e) => {
          if (e.key === "Enter" || e.key === " ") {
            e.preventDefault();
            setExpanded((prev) => !prev);
          }
        }}
        role="button"
        tabIndex={0}
      >
        <div className="flex items-center justify-between">
          <span className="text-sm font-medium text-secondary">모범 답안 참고</span>
          <svg
            width={16}
            height={16}
            viewBox="0 0 16 16"
            fill="none"
            className="text-muted transition-transform duration-200"
            style={{ transform: expanded ? "rotate(180deg)" : "rotate(0deg)" }}
          >
            <path
              d="M4 6L8 10L12 6"
              stroke="currentColor"
              strokeWidth={1.5}
              strokeLinecap="round"
              strokeLinejoin="round"
            />
          </svg>
        </div>
        {expanded && (
          <p className="mt-3 text-sm text-secondary leading-relaxed">{suggestedAnswer}</p>
        )}
      </div>
    )}

    {/* Start button */}
    <div className="flex justify-center">
      <button
        type="button"
        onClick={handleStart}
        disabled={!mediaReady}
        className={cn(
          "px-8 py-3 rounded-xl text-base font-semibold transition-all",
          mediaReady
            ? "bg-gradient-to-r from-indigo-500 to-purple-500 text-white hover:opacity-90 cursor-pointer"
            : "bg-card border border-white/[0.06] text-muted cursor-not-allowed",
        )}
      >
        연습 시작
      </button>
    </div>
  </div>
</div>
```

Note: The cleanup in useEffect must NOT stop tracks if `onStart` has been called (the stream is transferred to the drill engine). Use a `startedRef = useRef(false)` flag — set it in `handleStart`, and only stop tracks in cleanup if `startedRef.current === false`.
